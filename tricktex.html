NAME

tricktex - manual page for tricktex 0.6.0, 13.03.15  

SYNOPSIS

tricktex [,OPTIONS/] ,tex code/  

DESCRIPTION

Make a eps or ps file out of a tex code  

OPTIONS

-v, --version
Prints version
-h, --help
Show this help message
-p NAME
Add package
-o, --output FILE
Give the output filename. The default is 'myfile.eps'
--equ
Run in equation mode
--keep
Do not delete the tex file
--pic
Run in Picture mode
-t, --tight
tight bounding (uses bbbox_add.pl)
-w, --width NUMBER
Change textwidth of the pictures (implies --pic) The default is 1 (*\textwidth)
-r, --rotate
rotate pics by 90 degree
--view VIEWER
Run VIEWER at the end
 

EXAMPLES

tricktex --equ -o test.eps '\omega'
Create an omega symbol in test.eps
tricktex --equ -o test \\omega
Create an omega symbol in 'myfile.eps'
tricktex --equ -p color '\textcolor{blue}{\theta}'
Create a blue theta symbol in 'myfile.eps'
tricktex hallo
Create a "hallo" in 'myfile.eps'
tricktex --pic *.eps
Combine all eps picture into 'myfile.eps'
